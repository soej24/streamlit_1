데이터베이스에서 2차원 자료를 테이블이라고 한다.

변수이름 = 테이블 이름

ctrl + Enter 실행

use yh_db;  모모 디비에서 작업 하겠다.

항상 테이블 작업할때
id를 만든다.

PK = 데이터를 가져올때 빠르게 가져올수 있는!
NN = Not null (파이썬의 NaN 과 같은) 
체크되면 안에 값이 없으면 안된다는 뜻

AI = 오토 인크리먼트(Auto increment) (자동증가)
id는 음수가 나오면 안되기 때문에
un = Unsigned integer (언사이드 인티저)

id 컬럽 (PK, NN, UN, AI) 체크해야 한다.
